Random Go Stone Placements
==========================

> Generate a random starting position for your go games

Developing
----------

### Run ###

```bash
npm start
```

And view the site on <http://localhost:8000>.

### Test ###

```bash
npm test
```

or keep running and watch for changes:

```bash
npm run test:watch
```

### Lint ###

```bash
npm run lint
```

or (with auto-fix + format):

```bash
npm run lint:fix
```

### Type check ###

```bash
npm run check
```
